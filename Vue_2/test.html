校内二手物品交易平台

项目重构完成！已从Flask混合架构迁移到前后端分离架构。

本项目使用Flask作为后端API，Vue作为前端框架，实现了完整的校内二手物品交易功能。

后端 API

  • 基础URL: http://localhost:5000/api
  • 认证方式: JWT Bearer Token
  • 主要功能: 用户认证、商品管理、收藏、消息系统

前端应用

  • 访问地址: http://localhost:8080
  • 技术栈: Vue 3, Vue Router, Axios
  • 主要页面: 首页、商品详情、发布商品、个人中心、消息

如何启动

# 启动后端
cd backend
pip install -r requirements.txt
flask init-db
python run.py

# 启动前端
cd frontend
npm install
npm run serve
访问前端应用 查看API文档
